# Lynkit Clone Redesign

This project is a React-based clone of the [Lynkit.in](https://www.lynkit.in/) website, featuring a responsive design, video background, and transparent navigation. It also includes the original dashboard and trip view tasks, accessible via the "Products" dropdown.

## Project Overview

- **Landing Page**: Exact replica of Lynkit's hero section with video background.
- **Navigation**: Transparent navbar with dropdowns, matching the reference site's style.
- **Tech Stack**: React, Tailwind CSS (configured with Lynkit's color palette and typography).

## Tasks
The original tasks are preserved:
- **Task 1**: Landing Page (Home) - **Redesigned**
- **Task 2**: Analytics Dashboard (accessible via Products -> Analytics Dashboard)
- **Task 3**: Trip View (accessible via Products -> Trip View)

## Tech Stack
- **React.js**
- **Tailwind CSS** (Custom Configured)
- **Montserrat Font**

## How to Run
1. `npm install`
2. `npm run dev`
